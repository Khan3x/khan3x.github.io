Solution: #Boyer-Moore 多数投票算法

  1. 初始化：
    • candidate：候选元素，初始为 0。
    • count：计数器，初始为 0。
  2. 遍历数组：
    • 如果 count == 0，则将当前元素设为候选元素 candidate。
    • 如果当前元素等于 candidate，则 count 增加 1。
    • 如果当前元素不等于 candidate，则 count 减少 1。
  3. 返回结果：
    • 遍历结束后，candidate 就是多数元素。
